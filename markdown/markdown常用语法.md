# Markdown常用语法

+ **标题语法要在文字前面加上“# ”，n个“#”代表n级标题**

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
```

**显示效果如下**

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题

+ **字体加粗，需要在文字的两端加上两个*号**

```
原效果
**加粗效果**
```

**显示效果如下**

原效果
**加粗效果**

+ **斜体字，需要在文字两端加上一个*号**

```
原效果
*加粗效果*
```

**显示效果如下**

原效果
*加粗效果*
